# 개요

이 예제는 WPF 응용프로그램 내부에 간단한 REST API를 포함하는 방식을 보여준다.
UI에서 사용하던 장비 상태·제어 로직을 외부에서도 동일하게 접근할 수 있도록 하는 목적의 데모다.

# 왜 필요한가?

일부 장비·현장 시스템은 TCP/Serial만 제공하거나,
UI 프로그램 내부에만 데이터가 존재해 외부 연동이 어렵다.
이럴 때 WPF 내부에 **가벼운 API 서버**를 추가하면:

외부 모듈·웹서비스가 장비 상태를 조회할 수 있고

장비 제어를 외부 자동화 시스템에서도 수행할 수 있으며

기존 UI 코드를 그대로 활용해 API를 쉽게 확장할 수 있다.

# 사용 방법

프로그램을 실행하면 WPF UI와 함께 API 서버도 자동 시작된다.
